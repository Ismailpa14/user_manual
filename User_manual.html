1. About Software

Emission scanner Pro version 2.0 focuses on getting emission from the electronic devices and plot the heatmap on the desired emission place.

2. Installation

2.1 Application

Install the Emission scanner Pro software by running the setup file. Choose your preferred installation location.

2.2 Required connections

After the installation, make sure your connected the pc to camera, scanning device using micro USB cable, spectrum analyzer using USB-B and probe to spectrum analyzer using antenna cable.

3. Working

Steps for using the software:

  • Select Ports – Choose the camera port for scanning and select port for scanning device (Typically the port names like "USB-SERIAL CH340" ). After that click on "Home" button after that click on "Next" button.
  • Caliberarte probe – Select caliberation settings for probe caliberation (Setting -> configuration). then give a value to Z axis for the height.
  • Select the desired probe from main the window and click on Test probe button. After the probe reach on a point, click on red reload button to back on previous position.
  • Select BW – Select desired RBW and VBW values from the dropdown box, then click on "Set" button
  • Amplitude limit – Adjust the amplitude range on the y-axis (dB range).
  • Generate document – Create a document with the plotted graph.
  • Serial number – Assign a serial number for the graph in the document.
  • Average graph – The yellow graph represents the average from multiple files.
